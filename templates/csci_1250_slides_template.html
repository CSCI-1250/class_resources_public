üéØ Key Questions to Ask Yourself

  • What does this method do? Gets a string from the user
  • What does it need? A prompt to show the user
  • What does it give back? The string the user typed
  • How will I use it in Main? string name = GetUserStringInput("Enter your name: ");
Understanding Method Signatures: Example
static string GetCharacterType(int choice)

üîç Breaking Down the Signature

  • static - can call without creating object
  • string - returns a string value
  • GetCharacterType - descriptive method name
  • (int choice) - takes one integer parameter

üí° What This Method Should Do

  • Accept a number (1, 2, or 3)
  • Convert to character name
  • Return "Warrior", "Mage", or "Rogue"
  • Handle invalid input gracefully

Round 1: Implement High priority methods first (PrettyPrint, GetUserStringInput, GetUserIntInput, GetFortune) - these are essential for basic functionality

Round 2: Add Medium priority methods (DisplayMenu) - this improves user experience

Round 3: Build the main program loop connecting all methods together

Scaffolding means building your program step-by-step, testing each part before moving to the next. Think of it like constructing a building - you need a solid foundation before adding the roof!

Step 2: Implement Core
