What This Course Is About

📚 Course Description

Introduces all aspects of the programming and problem-solving process and the elements of good programming style. A high-level language is used for introducing these concepts. Laboratory use of the computer in designing, coding, debugging, and executing programs is an integral part of the course.

🎯 Primary Focus

  • Programming fundamentals
  • Problem-solving methodology
  • Good programming style
  • Hands-on coding experience

💻 Programming Languages

  • C# (C-Sharp) and Python
  • Modern, versatile language
  • Industry-standard
  • Great for learning fundamentals
What You'll Achieve

By the end of this course, you will be able to:

1

Foundation Knowledge

Understand foundational computer science topics and concepts

2

Career Preparation

Be prepared for a career in computing or related fields

3

Problem Solving

Apply a systematic framework for solving problems

4

Algorithm Development

Develop algorithms for solving simple problems

5

Produce solutions to simple problems in a managed programming language

6

Programming Fundamentals

Write code using Assignment, Input/Output, Selection, Iteration, and Methods

Programming Hackathons: Team Challenges

🏆 What's a Hackathon?

Team-based events where you apply course concepts to solve real programming problems in a collaborative, competitive environment.

👥 Team Process

  • Work in assigned groups
  • Break down the given problem
  • Design a solution approach
  • Implement the solution in code

📊 Evaluation Criteria

  • Group's written analysis
  • Team presentation quality
  • Program functionality
  • Code quality and style

🌟 Why Hackathons Matter

  • Practice real-world problem solving
  • Learn to work effectively in teams
  • Apply multiple course concepts together
  • Build communication and presentation skills
