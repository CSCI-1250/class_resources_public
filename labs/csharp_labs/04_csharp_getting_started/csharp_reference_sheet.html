C# Programming Fundamentals Reference Sheet

CSCI 1250 - Introduction to Computer Science and Programming

Basic Program Structure

Every C# program needs this structure: using System; public class Program { public static void Main(string[] args) { // Your code goes here Console.WriteLine("Hello, World!"); } } Key Points: • Everything must be inside a class • Main method is where execution starts • Case sensitive: Main (not main), Console (not console) • Every statement ends with semicolon ; • Code blocks use curly braces { }

Variables and Data Types

Major Difference from Python: In C#, you must declare the data type of every variable explicitly!
Variable Declaration Syntax: dataType variableName = value; Examples: string name = "Alice"; // String variable int age = 20; // Integer variable double grade = 85.7; // Decimal number variable bool isStudent = true; // Boolean variable Variable Lifecycle: // 1. Declaration - reserve memory space int studentAge; // 2. Initialization - first assignment studentAge = 20; // 3. Assignment - changing the value studentAge = 21; // 4. Combined declaration + initialization int maxStudents = 30;

Type Conversion

Automatic Widening (Safe): int smallNumber = 100; double bigNumber = smallNumber; // Automatic - no data loss Explicit Casting (Potentially Unsafe): double bigNumber = 100.7; int smallNumber = (int)bigNumber; // Result: 100 (decimal part lost) String to Number Conversion: string userInput = Console.ReadLine(); // Always returns string int age = int.Parse(userInput); // Convert to integer double price = double.Parse(userInput); // Convert to double Safe Parsing (Recommended): string input = Console.ReadLine(); if (int.TryParse(input, out int result)) { Console.WriteLine($"Valid number: {result}"); } else { Console.WriteLine("Invalid input!"); }

Input and Output

Console Output: Console.Write("Enter your name: "); // No newline Console.WriteLine("Welcome to C#!"); // With newline Console Input: string userInput = Console.ReadLine(); // Always returns string! Complete Input Example: Console.Write("Enter your age: "); string ageText = Console.ReadLine(); int age = int.Parse(ageText); Console.WriteLine($"You are {age} years old.");

String Formatting

String Interpolation (Recommended): string name = "Alice"; int age = 20; Console.WriteLine($"Hello, {name}! You are {age} years old."); Number Formatting: double grade = 87.6666; Console.WriteLine($"Your grade is: {grade:F2}%"); // Shows: 87.67% String Concatenation: string firstName = "Alice"; string lastName = "Smith"; string fullName = firstName + " " + lastName; Common Format Specifiers: {variable:F2} // 2 decimal places {variable:C} // Currency format {variable:P} // Percentage format

Control Flow - While Loops

Basic While Loop: int counter = 1; while (counter <= 3) { Console.WriteLine($"Count: {counter}"); counter++; // Same as counter = counter + 1 } User-Controlled Loop: string userInput = ""; while (userInput != "quit") { Console.Write("Enter 'quit' to stop: "); userInput = Console.ReadLine(); Console.WriteLine($"You entered: {userInput}"); }

Syntax Reminders

Critical Syntax Rules: • Every statement ends with semicolon ; • Use curly braces { } for code blocks • Parentheses required for method calls and conditions • Case sensitive: Console ≠ console, Main ≠ main • Use = for assignment, == for comparison • Comments: // single line, /* multi-line */ Common Syntax Errors: ❌ console.writeline("Hello") → ✅ Console.WriteLine("Hello"); ❌ string name = "Alice" → ✅ string name = "Alice"; ❌ while x < 10 → ✅ while (x < 10) { } ❌ if x = 5 → ✅ if (x == 5) { } ❌ name = "Alice" → ✅ string name = "Alice";

Building and Running C# Programs

Terminal Commands: dotnet new console -n ProjectName // Create new project cd ProjectName // Navigate to project dotnet run // Build and run dotnet build // Build only (check for errors) File Structure: ProjectName/ ├── Program.cs // Your code goes here ├── ProjectName.csproj // Project configuration └── bin/ // Compiled output (auto-generated)
Remember: Unlike Python where you can write code directly in a file, C# requires all code to be inside a class and method structure. Everything you write must go inside the Main method!
