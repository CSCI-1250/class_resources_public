Variables

Declaration and Initialization: variable_name = value # Declares AND initializes in one step Examples: name = "Alice" # String variable age = 20 # Integer variable grade = 85.7 # Float variable is_student = True # Boolean variable

✅ Legal Variable Names

student_name grade1 final_project my_variable _private_var userName (camelCase)

❌ Illegal Variable Names

2student # Can't start with number student-name # No hyphens student name # No spaces class # Reserved word def # Reserved word

Python Reserved Words (Keywords)

and as assert break class continue def del elif else except False finally for from global if import in is lambda None not or pass raise return True try while with yield Cannot use these as variable names!

Core Built-in Functions

input(prompt) - Gets user input as a string user_name = input("Enter your name: ") print(value) - Displays output to screen print("Hello World") print(variable_name) len(sequence) - Returns length of string/list length = len("hello") # Returns 5 type(variable) - Returns the data type print(type(42)) # <class 'int'>

String Formatting

F-Strings (Formatted String Literals): name = "Alice" age = 20 print(f"Hello, {name}! You are {age} years old.") Number Formatting: grade = 87.6666 print(f"Your grade is: {grade:.2f}%") # Shows: 87.67% Common Format Specifiers: {variable:.2f} # 2 decimal places {variable:>10} # Right align in 10 spaces {variable:<10} # Left align in 10 spaces

Functions

Function Definition Syntax: def function_name(parameter1, parameter2): """Optional docstring describing the function""" # Function body result = parameter1 + parameter2 return result Function Call: answer = function_name(5, 3) # Arguments: 5 and 3

Function Components

Component Description Example
Method Signature First line defining function name and parameters def calculate_grade(exam, hw):
Parameters Variable names in function definition exam, hw (in the definition)
Arguments Actual values passed when calling function 85, 92 (when calling)
Return Statement Sends value back to caller return final_grade
Function Body Indented code that runs when called All indented lines after colon

Control Flow - While Loops

Basic While Loop: counter = 1 while counter <= 3: print(f"Count: {counter}") counter = counter + 1 # Don't forget to update! User-Controlled Loop: user_input = "" while user_input != "quit": user_input = input("Enter 'quit' to stop: ") print(f"You entered: {user_input}")

Common String Methods

Method Description Example Result

.upper() Convert to uppercase "hello".upper() "HELLO"
.strip() Remove whitespace " hello ".strip() "hello"

Syntax Reminders

Important Syntax Rules: • Use colons (:) after if, while, for, def statements • Indentation matters! Use 4 spaces for each level • Parentheses required for function calls: print(), input(), float() • Case sensitive: myVariable ≠ myvariable ≠ MyVariable • Use = for assignment, == for comparison • Comments start with # symbol Common Syntax Errors: ❌ print "Hello" → ✅ print("Hello") ❌ if x = 5: → ✅ if x == 5: ❌ def myfunction → ✅ def myfunction(): ❌ while x < 10 → ✅ while x < 10:

Keep this reference handy while coding!
